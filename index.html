A Tag video é usada para adicionar um vídeo na página.

SRC

É o atributo usado para apontar o endereço do vídeo

Width

Usamos o width como atributo para definir uma largura para o vídeo.

Controls

O atributo controls tem a função de adicionar os controles para que o usuário consiga dar play, pause, adiante ou retroceder o vídeo. É muito importante adicionar o atributo controls.

Autoplay

É um atributo que play automaticamente no vídeo, alguns navegadores não permitem esse recurso, mas é possível deixar o vídeo com autoplay, adicionando outro atributo muted ou usando JavaScript.

Muted

O muted é um atributo que deixa o vídeo mudo, o usuário precisa usar o botão para conseguir ouvir o áudio do vídeo.

Loop

Deixa o vídeo em loop, quando o vídeo chegar ao fim, voltará para o começo.

Poster

O poster é usado para adicionar uma imagem de thumbnail, especificando o endereço da imagem.
